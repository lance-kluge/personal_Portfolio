Mini Git

A lightweight version control system built in C++ with commit tracking, file versioning, and a custom testing suite.

Project Overview

Mini Git was the final project for my CSCI 2275 Data Structures course in Fall 2023. My teammate and I built a lightweight version control system in C++ that tracked file versions, allowed reverting to previous commits, and included a robust testing suite.

Technical Details

We used a linked list to manage commits (each node being a commit) and a second linked list to track files within each commit, with hashes used for commit naming. Our testing suite included function tests and a comprehensive stress test that simulated all possible user actions, ensuring a stable and reliable final product.

Conclusion

Building Mini Git was an incredibly fun experience that enabled me to really start to understand working collaboratively . Ephram and I worked really well together and were able to develop a great product with a test suite that I am very proud of. We did so well that we even got a comment from our professor saying "Best submission I have seen. Code is clean and concise, well organized. Everything works exactly as expected. Great job!"
